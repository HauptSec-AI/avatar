# Avatar

This project is to be built entirely by Claude Code based on the specs here. `SPEC-AVATAR.md`
governs product behaviour and the backend (the text-chat digital twin, admin dashboard, auth,
Supabase, deployment); `SPEC-VOICE.md` is a companion spec that adds a voice channel on top of it.
When the two overlap, `SPEC-AVATAR.md` is the source of truth for shared infrastructure.
